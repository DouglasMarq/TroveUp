TroveUp

      

      
A Tauri port of TroveTools.NET, originally created by Dazo.

      
TroveUp allows easy installation of Trove modifications by searching and downloading from the
        Trovesaurus mod repository. This application creates the
        necessary override folders such that the files are not removed by the Glyph client when updating Trove.
