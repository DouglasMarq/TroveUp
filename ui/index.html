TroveUp

TroveUp is a mod manager for Trove. Browse, install, and keep your mods up to date directly from the
          Trovesaurus repository — no manual file wrangling
          required.

        
Mods are installed into override folders so they survive Glyph client updates. Create and share mod
          packs, and use the built-in modder tools to build and extract TMod files.
